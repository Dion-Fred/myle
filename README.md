# MYLE — My Learning Hub

A minimal, modern personal study platform built for deep focus and structured revision. Designed with a warm monochromatic brown aesthetic (creams, warm sands, muted taupes, and deep espressos), the system functions entirely on a local-first, client-side architecture that completely eliminates the need for external databases or authentication frameworks.

The platform utilizes a dynamic, decoupled configuration pipeline (tools.config.js) that reads standalone study tools directly from a dedicated directory. This architecture allows the main user interface to render seamlessly, updating available modules instantly and streaming target pathways automatically without requiring manual edits to the central dashboard file layout.
